# MVFLEX - Modern Construction Company Website

A premium, modern, responsive, and high-performance website for a professional Construction Company built with cutting-edge technologies.

## 🎯 Overview

MVFLEX is a full-featured construction company website featuring:
- Luxurious and trustworthy corporate design
- Minimal and elegant UI with premium animations
- Glassmorphism effects with subtle shadows
- Fully responsive design (desktop, tablet, mobile)
- SEO optimized with schema markup
- Fast performance and accessibility (WCAG)

## 🛠 Technology Stack

### Frontend
- **Next.js 14** - React framework with server-side rendering
- **TypeScript** - Type-safe JavaScript
- **Tailwind CSS** - Utility-first CSS framework
- **Framer Motion** - Advanced animations
- **React Hook Form** - Efficient form management
- **Swiper** - Touch slider component

### Backend
- **Node.js + Express.js** - Server runtime and framework
- **PostgreSQL** - Relational database
- **JWT** - Authentication & authorization

### Deployment
- **Vercel** - Optimized Next.js hosting

## 📋 Key Features

- Hero Section with animated statistics
- About Company with timeline
- 12 Service categories
- Portfolio with filters
- Testimonials slider
- Team members showcase
- Blog/News section
- Career opportunities
- Contact & inquiry management
- Dark/Light mode toggle
- Full responsiveness
- SEO optimization

## 🚀 Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Open http://localhost:3000
```
